# New Models Added to AvalAI Platform

**Date:** 2025-04-12

## Summary

We are excited to announce the addition of several new models to the AvalAI platform, expanding your options for various tasks including advanced reasoning, specialized transcription, search-integrated chat, and vision capabilities. This update includes models from XAI (Grok), OpenAI, Meta (via Together AI), and Alibaba (via Together AI).

---

## Details

The following models are now available through the AvalAI API:

### XAI (Grok)

*   **grok-3-beta**: High-performance model from XAI.
*   **grok-3-fast-beta**: Faster variant of Grok 3.
*   **grok-3-mini-beta**: Smaller, efficient Grok 3 variant.
*   **grok-3-mini-fast-beta**: Faster version of Grok 3 Mini.
*   *Documentation*: [XAI Models](en/providers/xai.md)

### OpenAI

*   **gpt-4o-transcribe**: Specialized model for high-accuracy audio transcription.
*   **gpt-4o-mini-transcribe**: Efficient variant for audio transcription.
*   **gpt-4o-search-preview / gpt-4o-search-preview-2025-03-11**: GPT-4o with integrated web search capabilities.
*   **gpt-4o-mini-search-preview / gpt-4o-mini-search-preview-2025-03-11**: GPT-4o mini with integrated web search capabilities.
*   *Documentation*: [OpenAI Models](en/providers/openai.md)

### Meta (via Together AI)

*   **llama-4-scout-17b-128e-instruct-fp8**: Experimental Llama 4 Scout model, potentially using FP8.
*   **llama-4-scout-17b-16e-instruct**: Experimental Llama 4 Scout model.
*   *Documentation*: [Meta Models (Llama)](en/providers/meta.md)

### Alibaba (via Together AI)

*   **qwen2.5-vl-72b-instruct**: Powerful Qwen model with vision capabilities.
*   **qwq-32b**: Capable text-based Qwen model.
*   *Documentation*: [Alibaba Models (Qwen)](en/providers/alibaba.md)

You can start using these models immediately via our standard API endpoints. Please refer to the linked documentation for specific model identifiers and usage examples.
