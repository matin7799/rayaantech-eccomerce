## Why AvalAI?

- **Zero-Markup Pricing** - Our prices are 100% based on the official pricing of the providers. You pay exactly what you would pay at the official provider - no hidden fees or markups.
- **One API, Multiple Providers** - Use OpenAI-compatible clients, official Anthropic SDKs, or Google's native GenAI SDK to access 26+ AI providers through a unified interface.
- **Triple SDK Support** - Choose between OpenAI-compatible SDKs (unified approach), Anthropic's official SDKs (native approach), or Google's GenAI SDK (native Gemini access).
- **Unified Schema** - Consistent request and response formats across all providers
- **Simple Integration** - No need to learn multiple APIs or manage different authentication methods.
- **Flexible Model Selection** - Easily switch between models from different providers without changing your code.
- **410+ AI Models** - Access the latest and most powerful AI models from leading providers.

```language-selector
bash=:curl https://api.avalai.ir/v1/responses \
  -H "Content-Type: application/json" \
  -H "Authorization: Bearer $AVALAI_API_KEY" \
  -d '{
    "model": "gpt-5.5",
    "instructions": "You are a helpful assistant.",
    "input": "Write a one-sentence bedtime story about a unicorn."
  }'

python=:import os
from openai import OpenAI

client = OpenAI(
    api_key=os.environ["AVALAI_API_KEY"],

go=:package main

import (
	"bytes"
	"encoding/json"
	"fmt"
	"io"
	"net/http"
	"os"
)

func main() {
	payload := map[string]any{
		"model":        "gpt-5.5",
		"instructions": "You are a helpful assistant.",
		"input":        "Write a one-sentence bedtime story about a unicorn.",
	}

	body, err := json.Marshal(payload)
	if err != nil {
		panic(err)
	}

	req, err := http.NewRequest("POST", "https://api.avalai.ir/v1/responses", bytes.NewBuffer(body))
	if err != nil {
		panic(err)
	}
	req.Header.Set("Authorization", "Bearer "+os.Getenv("AVALAI_API_KEY"))
	req.Header.Set("Content-Type", "application/json")

	resp, err := http.DefaultClient.Do(req)
	if err != nil {
		panic(err)
	}
	defer resp.Body.Close()

	responseBody, err := io.ReadAll(resp.Body)
	if err != nil {
		panic(err)
	}

	fmt.Println(string(responseBody))
}

    'instructions' => 'You are a helpful assistant.',
    'input' => 'Write a one-sentence bedtime story about a unicorn.',
];

$ch = curl_init('https://api.avalai.ir/v1/responses');
curl_setopt_array($ch, [
    CURLOPT_RETURNTRANSFER => true,
    CURLOPT_POST => true,
    CURLOPT_HTTPHEADER => [
        'Authorization: Bearer ' . $apiKey,
        'Content-Type: application/json',
    ],
    CURLOPT_POSTFIELDS => json_encode($payload),
]);

$response = curl_exec($ch);
curl_close($ch);

echo $response;

```

## Retrieve Available Models

You can retrieve a list of all available models and their metadata using the Models API:

```bash
# Public endpoint (no authentication required)
curl https://api.avalai.ir/public/models

# Authenticated endpoint (includes your tier-specific rate limits)
curl https://api.avalai.ir/v1/models \
  -H "Authorization: Bearer $AVALAI_API_KEY"

# Get detailed info for a specific model (includes pricing, rate limits, capabilities)
curl https://api.avalai.ir/v1/models/gpt-5.5 \
  -H "Authorization: Bearer $AVALAI_API_KEY"
```

The retrieve endpoint returns detailed information including:
- **Metadata** - Model capabilities (vision, function calling, PDF support, etc.)
- **Pricing** - Per-million-token costs (input, cached input, output)
- **Rate Limits** - Your current tier limits and all tier limits

curl https://api.avalai.ir/user/credit \

**Sample Response:**

```json
{
  "limit": 0.0,
  "remaining_irt": 800000.0,
  "remaining_unit": 0.0,
  "total_unit": 10.0,
  "exchange_rate": 80000
}
```

**Response Fields Explanation:**
- `remaining_irt` - Remaining credit in IRT
- `remaining_unit` - Remaining credit converted to units (based on Tether exchange rate)
- `exchange_rate` - Real-time currency conversion rate
- `total_unit` - Total credits including bank payments, gift credits, grants, etc., based on real-time conversion rate
